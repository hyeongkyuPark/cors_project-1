## 책 중고 마켓 프로젝트

### 사용 기술

### server
- NCP (Naver Cloud Platform)
  - `CentOS` 기반의 micro server 사용.
  - `volta` (javascript manager) tool로 node,pm2 등 버전관리
  - proxy-server : `nginx`
  - RDB : `Mysql` ver 8.0.23
  - SSL : `certbot`

## back-end
- Programming Language : `java ` ver 11
- Framework : `springboot` ver 2.4.2
  - Embedded WAS : `tomcat`
- ORM
  - `JPA`
    - Hibernate
    - `Spring Data JPA`
  - `QueryDsl`
- `Spring security`
- Stomp (프론트 미구현)
